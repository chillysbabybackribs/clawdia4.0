Where should identity profile settings live?

Pick the placement that feels most natural in Clawdia

A

Section in existing Settings

Add an "Identity" section to the current Settings view — same back-button pattern, same scrollable layout. No new navigation needed.

B

Dedicated Identity view

A separate full-screen view reachable from Settings or a dedicated icon in the rail. Gives identity its own space as the autonomy layer grows.

C

Modal / overlay

A centered modal triggered from anywhere (Settings button, chat suggestion, etc.). Lightweight, doesn't navigate away from current context.
