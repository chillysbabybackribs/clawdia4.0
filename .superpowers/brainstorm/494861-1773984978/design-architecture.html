Architecture Design

AI ↔ Calendar integration — full system overview

Data & Control Flow
AI (shell_exec via CLIanything)
↓ calls: clawdia-cal add "Team standup" 2026-03-21 10:00 60
↓ calls: clawdia-cal list --date 2026-03-21
↓ calls: clawdia-cal delete <id>

clawdia-cal CLI (new: scripts/clawdia-cal)
↓ reads/writes → ~/.config/Clawdia/calendar.sqlite
(separate file from data.sqlite — clean isolation)

Electron Main Process
↓ fs.watch on calendar.sqlite → debounce 150ms
↓ reads events → pushes via IPC: calendar:events-changed

Calendar.tsx (renderer)
↓ listens on window.clawdia.calendar.onEventsChanged
↓ renders event dots on calendar days
↓ shows event list on selected date
