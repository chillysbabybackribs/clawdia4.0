Live Update & AI Context Injection

How changes flow from CLI → UI, and how the AI stays aware

Live update sequence
1. AI calls: clawdia-cal add "Standup" --date 2026-03-21 --time 10:00 --duration 30
→ CLI writes row to calendar.sqlite
→ CLI exits with JSON output (AI reads it)

2. calendar-watcher.ts detects file change
→ fs.watch fires on calendar.sqlite
→ debounce 150ms (handles double-fire)
→ reads all events for visible month range
→ mainWindow.webContents.send('calendar:events-changed', events)

3. Calendar.tsx receives push
→ window.clawdia.calendar.onEventsChanged(cb)
→ setEvents(events) → re-render dots + list
Total latency: <200ms from CLI exit to UI update
