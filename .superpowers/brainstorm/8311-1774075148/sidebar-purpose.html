B — Mode Tabs + "Try Asking"

Sidebar has three tabs: Chat (history), Agents (live processes), Tasks (swarm view). Below the tabs: a rotating "Try asking..." section with example prompts that show off capabilities — clickable to prefill the input. Teaches by example.

C — Icon Rail + Capability Drawer

Thin icon rail (44px) with mode icons: Browser, Files, Desktop, Agents, Settings. Clicking an icon slides out a 160px drawer explaining that capability with examples and the slash command to activate it. Collapses back when you click the chat area.

These aren't mutually exclusive — elements can be combined. Pick what resonates and describe what you'd change or add.
